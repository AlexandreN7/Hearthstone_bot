Hearthstone_bot dependencies
*Python2
* OpenCV
*pip2.7 -> packet named python2_pip
* Python autopy // keyboard & mouse control
    $sudo pip2.7 install autopy
* Python numpy // mostly math
    $sudo pip2.7 install numpy

Note:
You have to put the deck you want to play in deck_bot.txt (english)
You have to put the game in 1024x768
